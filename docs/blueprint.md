# **App Name**: Mis Dulces Quince

## Core Features:

- Revelación Animada de Invitación: Una página de inicio interactiva que se despliega o hace una transición con gracia para revelar los detalles completos de la invitación, simulando la apertura de una invitación física.
- Reproductor de Música de Fondo: Un reproductor de música integrado con controles (reproducir/pausa) para una canción de fondo seleccionada, estableciendo el ambiente de celebración para los invitados.
- Galería de Fotos y Videos: Una galería de fotos y videos desplazable que muestra recuerdos preciados de la Quinceañera, completa con subtítulos.
- Cuenta Regresiva para la Celebración: Un temporizador de cuenta regresiva dinámico que muestra los días, horas y minutos restantes hasta que comience la celebración de la Quinceañera.
- Enlaces Interactivos de Mapas de Ubicación: Enlaces directos o mapas integrados para la iglesia y los lugares de la fiesta, lo que permite a los invitados obtener fácilmente indicaciones a través de su servicio de mapas preferido.
- Botón de Contacto de WhatsApp: Un botón prominente y fácil de usar para conectar instantáneamente a los invitados con el anfitrión para confirmaciones de asistencia o consultas a través de WhatsApp.
- Herramienta de Redacción de Eventos con IA: Una herramienta de IA que sugiere frases elegantes y poéticas para varias secciones de la invitación (por ejemplo, mensajes de bienvenida, descripciones de eventos), lo que ayuda al anfitrión a crear el texto perfecto. Usa una herramienta para decidir si es necesario este texto.

## Style Guidelines:

- Color primario: Un elegante rosa (#FF69B4) que representa la juventud y la alegría.
- Color secundario: Blanco (#FFFFFF) para crear un fondo limpio y sofisticado.
- Color de acento: Rojo (#FF0000) para resaltar elementos importantes y llamadas a la acción.
- Fuente para titulares: 'Great Vibes', una fuente de script elegante que transmite sofisticación y tradición.
- Fuente para el cuerpo del texto: 'Montserrat', una fuente sans-serif moderna y legible para garantizar la claridad.
- Utilizar iconos delicados y artísticos, posiblemente con motivos florales o de filigrana sutiles, para complementar el tema elegante y festivo sin sobrecargar el diseño.
- Un diseño de desplazamiento inmersivo de varias secciones donde cada sección está claramente delineada y guía al invitado a través del viaje de la invitación, inspirado en elegantes diseños de revistas digitales.
- Animaciones sutiles y fluidas para transiciones de página, revelaciones de elementos (como la apertura inicial de la invitación) y estados de desplazamiento, que agregan un toque de encanto y sofisticación moderna.
